# Quiz Cards
Diese Applikation ermöglicht Ihnen Quizkarten zu erstellen. Diese dann nach wahl bearbeiten und Löschen. Um die Quizkarten üben zu können kann man das Resultat verstecken oder anzeigen lassen.
